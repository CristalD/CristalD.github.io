## 🚀 Cómo subir a GitHub Pages

### 1. Crea un repositorio en GitHub

1. Ve a [github.com/new](https://github.com/new)
2. Nombre del repositorio: `portfolio` (o `cristal-portfolio`)
3. Visibilidad: **Public** ✅
4. **No** inicialices con README (ya tenemos uno)
5. Haz clic en **Create repository**

### 2. Sube el código

Abre tu terminal en la carpeta del proyecto y ejecuta:

```bash
git init
git add .
git commit -m "🚀 Initial portfolio"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/portfolio.git
git push -u origin main
```

> Reemplaza `TU_USUARIO` con tu nombre de usuario de GitHub.

### 3. Activa GitHub Pages

1. Ve a tu repositorio en GitHub
2. Haz clic en **Settings** (⚙️)
3. En el menú lateral, busca **Pages**
4. En **Source**, selecciona **GitHub Actions**
5. ¡Listo! El workflow se ejecutará automáticamente

### 4. Accede a tu portafolio

Tu portafolio estará disponible en:

```
https://TU_USUARIO.github.io/portfolio/
```

El deploy tarda ~1-2 minutos. Puedes ver el progreso en la pestaña **Actions** de tu repositorio.

## 📝 Personalizar

Para cambiar el color de acento principal, edita en `styles.css`:
```css
--accent: #7c6af7;
```
